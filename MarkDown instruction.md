# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивлм необходимо обрамить его звездочками (*). Например, *вот так*

Чтобы выделить текст полужирным, необходимо обрамить его двойными зведочками (**).
Например, **вот так**.
